# Archaeohack Visual Translator App for Ancient Egypt Language

# designed by 3A1W, members: Mazen Henderson, Hansel Chen, Kevin Liu, Henry Wang

Archaeohack Visual Translator App is a Python driven that allows archaeologists to hand-draw ancient Egyptian signs, which are then recognized and classified by a visual neural network trained on hieroglyphic forms.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the Archaeohack Visual Translator App.

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
